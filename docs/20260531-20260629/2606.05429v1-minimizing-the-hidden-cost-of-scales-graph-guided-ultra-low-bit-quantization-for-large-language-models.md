## 摘要
训练后量化（Post-training quantization, PTQ）对于大语言模型（LLMs）的高效部署至关重要。最近的超低位PTQ方法依赖于刚性的权重显著性假设或位置启发式，引入了大量隐藏的缩放参数开销。我们提出了SAGE-PTQ（显著性感知图引导高效PTQ），一种新颖的LLM超低位量化框架，能够最小化隐藏的缩放成本。SAGE-PTQ利用分布统计将显著权重和非显著权重分离，然后将子采样的非显著权重建模为稀疏图，以估计每层的最佳分组数。SAGE-PTQ采用双模式量化，为显著权重分配多位精度，并将非显著权重二值化。为了减少缩放参数开销，SAGE-PTQ对显著权重使用每个通道一个缩放因子，对每个非显著组使用一个标量。最后，SAGE-PTQ实现自适应显著性阈值，以选择每个矩阵的最佳显著性比率。SAGE-PTQ平均每个矩阵达到1.03权重比特和仅0.004缩放比特，优于BiLLM和PB-LLM等最先进方法。在LLaMA-3-8B上，SAGE-PTQ实现了6.74的WikiText2困惑度，而BiLLM为55.8，同时使用的GPU内存不到BiLLM的50%。在LLaMA-2-70B上，SAGE-PTQ在单个NVIDIA L40 GPU上提供了1.5倍更快的解码速度，展示了实际的推理效率。

## Abstract
Post-training quantization (PTQ) is critical for the efficient deployment of large language models (LLMs). Recent ultra-low-bit PTQ methods rely on rigid weight-saliency assumptions or position heuristics, introducing substantial hidden scaling overhead. We propose SAGE-PTQ (Saliency-Aware Graph-guided Efficient PTQ), a novel ultra-low-bit quantization framework for LLMs that minimizes hidden scaling cost. SAGE-PTQ separates salient and unsalient weights using distributional statistics, then models subsampled unsalient weights as a sparse graph to estimate the optimal number of groups per layer. SAGE-PTQ applies dual-mode quantization, assigning multi-bit precision to salient weights and binarizing unsalient weights. To reduce scaling overhead, SAGE-PTQ uses one per-channel scale for salient weights and one scalar per unsalient group. Finally, SAGE-PTQ implements adaptive saliency thresholding to select the optimal saliency ratio per matrix. SAGE-PTQ achieves 1.03 weight bits and only 0.004 scaling bits per matrix on average, outperforming state-of-the-art methods such as BiLLM and PB-LLM. On LLaMA-3-8B, SAGE-PTQ achieves 6.74 WikiText2 perplexity, compared to 55.8 for BiLLM, while using less than 50% of BiLLM's GPU memory. On LLaMA-2-70B, SAGE-PTQ provides 1.5x faster decoding on one NVIDIA L40 GPU, demonstrating practical inference efficiency.

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

**核心思想**：  
将权重矩阵分为**显著（salient）** 和**非显著（unsalient）** 两部分。显著权重（少量但敏感）保持多位精度，非显著权重（大量但不敏感）进行二值化。通过图引导算法自适应确定非显著权重的最优分组数，并采用单标量缩放因子以最小化缩放开销。

**关键技术细节**：

1. **显著性感知权重划分**：
   - 基于权重大小（magnitude）作为显著性指标（经验证明优于Hessian指标）。
   - 每层权重矩阵按如下规则划分显著集Ω_sal：  
     `Ω_sal = {w ∈ Ω | |w| > μ + σ·Φ⁻¹(ζ_sal/2)}`  
     其中 μ、σ 为均值、标准差，Φ⁻¹为probit函数，ζ_sal为可学习阈值。

2. **图引导的非显著组数估计**：
   - 对非显著权重进行随机子采样（0.03%），构建稀疏KNN图。
   - 应用谱聚类，并使用轮廓系数（Silhouette Score）评估候选组数 r，选择使轮廓系数最大的组数 N_opt。
   - 最终将非显著权重按分位数均匀分为 N_opt 组，每组用一个标量缩放因子。

3. **双模量化**：
   - **模式1（显著权重）**：使用单位通道尺度（as_per_channel）和多位量化（例如2~4比特），交替优化尺度与量化值。
   - **模式2（非显著权重）**：每组使用单一标量尺度，进行三元（{-1,0,1}）二值化，公式如下：  
     `b_ij* = sign(w_ij) if a > 0 else -sign(w_ij)`；`a* = <W,B> / ||B||²`。

4. **自适应显著性阈值**：
   - 对每一权重矩阵，求解优化问题最小化归一化重构误差，使用Brent方法找到最优 ζ_sal，受预设预算上限约束。

5. **高效推理运行时**：
   - 预计算每个权重组索引的查找表（LUT），使用比特打包存储于CPU内存，推理时按层动态加载，避免GPU内存额外开销。

**算法流程（文字说明）**：
- 输入：预训练权重矩阵 W。
- 步骤①：基于分布统计分离显著权重与非显著权重。
- 步骤②：子采样非显著权重，构建KNN图，经谱聚类和轮廓系数确定最优组数 N_opt。

- 步骤④：对显著权重执行多位量化（交替优化+离散化），对每组非显著权重执行三元二值化。
