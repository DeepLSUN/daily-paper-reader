## 摘要
随着大语言模型（LLM）数据集规模扩展至数万亿词元，数据选择已成为过滤无信息噪声并构建自适应学习轨迹的关键前沿。除了静态启发式过滤外，用于LLM训练的高级数据选择方法主要遵循两种范式，每种都有根本性缺陷。基于影响的方法提供了原则性的双层目标，但需要难以处理的逆Hessian计算，而超额损失方法计算高效，但依赖于一个静态参考模型，该模型在训练过程中会与不断演化的代理模型失配。我们提出BLADE（双层自适应数据选择），一种无Hessian的数据选择框架。BLADE通过拉格朗日乘子将基于影响的方法背后的双层优化问题重新表述为带惩罚的单层目标，避免了逆Hessian计算，同时揭示了与基于超额损失的数据选择之间的原则性联系。所得目标恢复了超额损失形式，但用与训练保持同步的动态参考模型取代了静态参考模型。理论上，我们证明了该惩罚形式保证了一阶收敛。为了实现高效的在线批次选择，我们将BLADE实例化为一种无记忆随机块坐标Frank-Wolfe算法。大量实验表明，BLADE持续优于最先进的数据选择基线，为LLM训练提供了实用方案。

## Abstract
As Large Language Model (LLM) datasets scale to trillions of tokens, data selection has emerged as a critical frontier to filter out uninformative noise and construct adaptive learning trajectories. Beyond static heuristic filtering, advanced data selection methods for LLM training largely follow two paradigms, each with fundamental limitations. Influence-based methods provide principled bi-level objectives but require intractable inverse-Hessian computations, while excess-loss methods are computationally efficient but rely on a static reference model that becomes misaligned with the evolving proxy model during training. We propose BLADE (Bi-Level Adaptive Data sElection), a Hessian-free framework for data selection. BLADE reformulates the bi-level optimization problem underlying influence-based methods as a penalized single-level objective via Lagrange multipliers, avoiding inverse-Hessian computation while revealing a principled connection to excess-loss based data selection. The resulting objective recovers an excess-loss form but replaces the static reference model with a dynamic one that stays synchronized with training. Theoretically, we prove that this penalized formulation guarantees first-order convergence. For efficient online batch selection, we instantiate BLADE as a memoryless randomized block-coordinate Frank-Wolfe algorithm. Extensive experiments show that BLADE consistently outperforms state-of-the-art data selection baselines, providing a practical recipe for LLM training.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：大语言模型（LLM）预训练数据集已扩展至数万亿 token，但其中包含大量低质量、噪声或冗余数据，直接训练会导致收益递减，甚至损害模型推理能力。因此，如何从海量候选数据中自动选择信息密度高、对模型学习有益的 token 成为关键挑战。
- **现有方法局限**：
  - **基于影响（Influence-based）的方法**（如 LESS、MATES）理论上通过对验证损失的影响进行评分，但需要计算逆 Hessian 矩阵，计算量巨大，难以扩展到 LLM 规模。
  - **超额损失（Excess-loss）的方法**（如 RHO-1）通过代理模型与静态参考模型的损失差来评分，计算高效，但参考模型在训练前固定，随着训练中代理模型演化，参考信号会逐渐过时，导致选择质量下降。
- **本文目标**：提出 BLADE，一种无 Hessian 的双层数据选择框架，融合两类方法的优点：既保留影响方法的理论严谨性，又具备超额损失方法的可扩展性，同时引入动态参考模型解决信号滞后问题。

## 2. 方法论：核心思想、关键技术细节与算法流程

- **核心思想**：将数据选择建模为双层优化问题（外层最小化验证损失，内层在加权训练损失上优化模型参数），通过拉格朗日乘子转化为带惩罚的单层目标，避免逆 Hessian 计算。该惩罚目标自然恢复为超额损失形式，但参考模型不再是静态的，而是与训练中的代理模型周期性同步。
- **关键技术细节**：
  - **双层优化形式化**：定义连续选择向量 α（每个 token 的权重），约束 α ∈ [0,1]^N 且 L1 范数固定（保持率 γ）。外层目标 min_α L_val(w(α))，w(α) 是内层加权损失的最小化器。
  - **惩罚转化**：引入惩罚参数 λ，将内层最优性条件作为惩罚项加入外层，得到单层目标 H_λ(α, w) = L_val(w) + λ[L_train(α, w) - min_u L_train(α, u)]。理论上当 λ→∞ 时，解逼近原双层问题。
  - **三变量更新**：
    - **代理模型 u**：在每个 batch 上以选中的 token 加权训练一步（公式(3)）。
    - **参考模型 w**：每 τ 步（一个 episode）从代理模型初始化，在验证数据和训练数据上同时梯度下降 K 步（公式(4)），然后用于后续 τ 步的数据评分。
    - **数据权重 α**：根据当前 proxy 和 reference 的 token 级损失差 Δ_n = L_train^n(reference) - L_train^n(proxy) 进行 Top-γ 二值选择（公式(6)），该选择通过无记忆随机块坐标 Frank-Wolfe 算法在线实现。
  - **在线批处理**：每次迭代从候选集中随机采样一个 batch，在该 batch 内按损失差排序选取 Top-γ 的 token，无需全局评分，适应非平稳训练动态。
  - **保持上下文结构**：未被选中的 token 不物理删除，而是通过 loss mask 掩蔽其损失项，保证自回归依赖完整。
- **理论保证**：在 PL 条件、光滑性、有界 Hessian 等标准假设下，证明 BLADE 以 O(T^{-1/4}) 的速率收敛到原双层问题的一阶驻点（Frank-Wolfe Gap 衡量）。

## 6. 主要结论与发现

1. **性能提升**：在领域迁移（数学）上，BLADE 在 TinyLlama-1.1B 上平均准确率 29.7%，超过 RHO-1（25.7%）4 个百分点，超过 Base-CT 13 个百分点；在 LLaMA-2-7B 上平均 43.8%，超过 RHO-1（41.1%）。在通用预训练上，平均 51.73%，超过 RHO-1（50.58%）。
2. **动态参考的关键作用**：静态参考（RHO-1）的损失差分布随训练趋于零或负值，失去判别力；BLADE 通过动态同步保持正损失差（Δ > 0）的右尾，持续选择高边际收益的 token。
3. **超参数鲁棒性**：参考更新间隔 τ 在 500~2000 之间性能稳定，可降低计算开销。
4. **同步机制必要性**：无同步时参考模型与代理模型距离发散，导致惩罚项不稳定；同步后距离有界，保证训练稳定。
5. **理论收敛保证**：证明 BLADE 能一阶收敛到双层目标驻点，速率 O(T^{-1/4})。

## 7. 优点

- **理论统一性**：从双层优化出发，通过惩罚方法桥接了基于影响与超额损失两种范式，既有严谨理论基础又实用。
- **计算高效性**：完全避免 Hessian 计算，仅需前向传播计算损失差，复杂度与 RHO-1 接近（约 2 倍于普通训练）。参考模型离线预计算可进一步优化。
- **在线自适应**：动态参考模型随训练同步，持续校准选择信号，比静态方法更灵活。
- **保持上下文完整性**：通过 loss mask 而非物理删除 token，不破坏自回归因果关系。
- **实验设计全面**：涵盖领域和通用场景，多种模型尺寸，充分的消融分析，并测试了可迁移性。

## 8. 不足与局限

- **模型规模受限**：最大实验到 7B 参数，未在百亿/千亿级模型上验证。论文自身承认“由于学术计算资源限制”。

- **超参数依赖**：保持率 γ 需手动设定（论文建议 0.6），虽经分析但不同任务可能需调整。参考模型内循环步数 K 也未充分探索（固定 300）。
- **计算开销仍高于无选择训练**：BLADE 比 Base-CT 耗时约 50% 以上，虽可接受但未完全“免费”。
- **选择偏差风险**：强调选择高损失差的 token 可能偏好“硬”样本，若验证集本身有偏差可能放大。
- **跨模型迁移性有限**：小模型选出的 token 对大模型帮助不够大，限制了跨架构复用能力。
- **未与其他方法（如 DSIR、D4、GBFT）对比**：仅对比了 MATES 和 RHO-1，覆盖面可进一步扩展。
