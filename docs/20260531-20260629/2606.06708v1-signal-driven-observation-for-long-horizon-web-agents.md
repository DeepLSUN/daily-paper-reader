## 摘要
在长周期运行的网络代理中，每个动作步骤都会读取原始DOM树和可访问性树——通常包含数万个标记——导致上下文逐步退化，使得代理在任务完成前推理能力便已下降。我们认为，这种将观察频率与动作频率耦合的方式是一种架构性错误。受递归语言模型中查询文档优于整体阅读这一见解的启发，我们提出了信号驱动观察（Signal-Driven Observation, SDO）：一个专用的子调用读取完整DOM，但只返回与任务相关的元素及其选择器，并且仅当轻量级信号检测器被触发时才会重新调用该子调用——触发信号包括URL变化、新出现的可交互元素、动作失败或外部浏览器事件。我们概述了SDO所引入的开放问题，并呼吁社区将观察压缩视为网络代理设计中的核心架构决策。

## Abstract
Web agents operating over long horizons ingest raw DOM and accessibility trees -- routinely tens of thousands of tokens -- at every action step, causing progressive context degradation that erodes reasoning well before tasks complete. We argue that this coupling of observation frequency to action frequency is an architectural mistake. Drawing on the insight from Recursive Language Models that querying a document outperforms reading it wholesale, we propose Signal-Driven Observation (SDO): a dedicated sub-call reads the full DOM but returns only task-relevant elements and their selectors, and is re-invoked only when a lightweight signal detector fires -- triggered by URL transitions, newly visible interactive elements, action failures, or exogenous browser events. We outline the open problems SDO introduces and call on the community to treat observation compression as a core architectural decision in web agent design.

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

- **核心思想**：受递归语言模型（RLMs）启发——对于大型静态文档，通过查询程序化提取所需信息比整体读取效果更好。将这一思想应用于动态网页DOM：专用子调用（sub RLM）读取完整DOM，但仅返回任务相关元素及其选择器；轻量级信号检测器决定何时需要重新观察。
- **四个组件**：
  - **Root LM（根模型）**：维护任务规范T、紧凑观察序列O1..Ot、动作历史H。从不读取原始DOM，规划动作序列并仅在收到新观察时重新规划。
  - **sub RLM（子调用）**：专注DOM观察，读取完整可访问性树，返回结构化列表（元素目的、类型、当前值、Playwright选择器）。不执行动作，只回答“页面上哪些内容对任务重要，在哪里”。
  - **信号检测器（Signal Detector）**：零LLM成本，在每次Playwright动作后检查四个条件：URL跳转、新ARIA元素出现（如列表、对话框）、动作失败、外部事件（如cookie横幅、AJAX更新）。
  - **浏览器**：执行根模型的Playwright动作序列，动作后控制交给信号检测器。
- **算法流程**：
  - 任务开始：初始页面加载触发sub RLM调用，产生O1；根模型接收(T, O1, H=[])并规划初始动作序列S。
  - 对每个动作a∈S：浏览器执行a，动作追加到H，信号检测器检查四个条件。
    - 若无信号：立即执行下一动作（无LLM调用，无DOM读取）。
    - 若有信号：调用sub RLM产生Ot+1，根模型接收更新(T, Ot+1, H)并重新规划。
- **核心贡献**：解耦观察频率与动作频率，仅在环境状态有意义改变时才触发重新观察，从而防止上下文退化和目标漂移。
