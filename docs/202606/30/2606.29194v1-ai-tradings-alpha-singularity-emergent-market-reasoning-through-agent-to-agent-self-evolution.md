## 摘要
自动化阿尔法挖掘保持评分函数固定，并在此之上变化搜索算法。针对固定评分器收敛的搜索会过度拟合评分器无法惩罚的任何因素，这是样本外泛化差距的主要原因。我们将评分函数视为与阿尔法因子并列的搜索产物，并研究何种条件能使这种联合搜索可行。密封联合搜索（SJS）是一个框架：它规定了自主发现系统中信息流的一组结构条件，这些条件防止联合搜索坍缩为自我确认，同时保持评估器密封。条件涵盖角色分解、类型化角色间通信、来源密封读取、版本化存储以及底层局部提升。Agora对SJS进行了实证测试：五类LLM智能体通过三个通道通信，进化出八个技能库，其中阿尔法库基于AlphaGen算子构建。三名评估器撰写报告汇总为一份简报，传递分歧而非投票。我们在CSI 1000上运行Agora 100轮，并在一个从所有LLM输入密封的91天2026年保留集上进行评估。Agora实现了保留集夏普比率+1.87；最佳基线在有利种子下为+1.334，跨种子均值为-0.755。将Agora的两个指标预加载到冻结库消融中仅恢复了+2.25夏普差距中的+0.40，而添加PPO但不进行库进化则加剧了差距。这两个指标是涌现出来的，而非设计的。注意事项：单种子运行，信号集中于空头侧，设计用于多空策略。

## Abstract
Automated alpha mining holds the scoring function fixed and varies the search algorithm over it. A search that converges against a fixed scorer overfits whatever the scorer cannot penalize, a primary cause of the out-of-sample generalization gap. We treat the scoring function as a search artifact alongside the alpha factors and study what conditions make this joint search admissible. Sealed Joint Search (SJS) is a framework: a set of structural conditions on information flow in an autonomous-discovery system that prevent joint search from collapsing into self-confirmation while keeping the evaluator sealed. Conditions cover role decomposition, typed inter-role communication, provenance-sealed reads, versioned stores, and substrate-local promotion. Agora tests SJS empirically: five LLM agent classes communicate via three channels, evolving eight skill libraries, with alpha libraries built on AlphaGen operators. Three evaluators write reports aggregated into one brief, carrying forward disagreement instead of voting. We run Agora for 100 rounds on CSI 1000 and evaluate on a 91-day 2026 holdout sealed from all LLM inputs. Agora achieves holdout Sharpe +1.87; best baseline +1.334 at favorable seed and -0.755 cross-seed mean. Pre-loading Agora's two metrics into a frozen-library ablation recovers only +0.40 of the +2.25 Sharpe gap, and adding PPO without library evolution worsens the gap. The two metrics emerge rather than being designed. Caveats: single-seed run, short-side concentrated signal, intended for long-short.

# 论文总结：《AI Trading's Alpha Singularity: Emergent Market Reasoning through Agent-to-Agent Self-Evolution》

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：在自动化因子挖掘（alpha mining）中，传统的搜索方法（如遗传规划、强化学习）保持评分函数固定，仅变化搜索算法。这导致搜索过程会过度拟合评分函数无法惩罚的方面，造成严重的样本外泛化差距。
- **研究动机**：将评分函数本身视为搜索工件（artifact），让搜索同时优化因子表达式和评分函数，从而避免固定评分器的盲点。然而，同时搜索两者会引发“自我确认”坍缩：搜索过程可以随意放松评分函数来接纳任何候选解。
- **整体含义**：为解决这一坍缩，论文提出**密封联合搜索（Sealed Joint Search, SJS）**框架，从信息流拓扑层面约束系统结构，使得联合搜索在不依赖外部评估器频繁干预的情况下仍能产生可泛化的解。

## 2. 方法论

### 核心思想
- SJS 是一组关于自主发现系统中信息流的结构条件（F1–F5），而非具体的优化算法。
- 关键重新定位：泛化性能由搜索的**信息拓扑**决定，而非任何参与者的局部行为。通用性分析成为可组合的：只要满足SJS条件，任何尊重边界的局部更新规则都会继承拓扑的泛化性质。

### 关键技术细节
- **五个结构条件（F1–F5）**：
  - **F1 分解提案**：提案者与评估者分离，无共享状态。
  - **F2 类型化角色间通信**：禁止自由形式的聊天；所有跨角色信息通过带类型标签的通道传输。
  - **F3 来源密封底板**：每条记录带有数据段来源标签；读取时按角色和内容类型过滤。
  - **F4 持久化版本化工件存储**：每个角色拥有自己的技能库，状态机为 builtin / trial / accepted / rejected。
  - **F5 底板局部提升规则**：非自判定、基于结果的提升（例如，基于训练夏普比率的相关性自动提升）。
- **边界条件**：外部评估器在运行前固定，所有LLM输入无法访问。

## 6. 主要结论与发现

- **Agora表现**：保留段组合夏普**+1.87**，高于所有基线。最佳单种子B2为+1.334；B2跨种子均值仅为**-0.755**（标准差2.20）。

## 7. 优点

### 方法论创新
1. **SJS框架**：将泛化问题从搜索算法转移到信息拓扑，提供可组合的泛化保证。该框架通用，可适用于其他自主发现领域（定理证明、程序合成等）。

3. **评估面板机制**：三个评估者各自撰写报告，底板聚合保持分歧而不是投票，避免信息损耗。

## 8. 不足与局限

### 实验覆盖
- **单种子系统性风险**：Agora仅运行一次，而基准的种子方差极大（B2的4.4夏普点），全系统的种子方差完全未知。头榜+1.87可能只是幸运样本。
- **有限保留窗口**：仅91天（约18周），Newey-West检验无法区分大多数对比；更长的保留集（如1-2年）是必要的。
- **缺少单智能体进化基线**：未比较“一个LLM同时提案因子和指标”的情况，导致F1（角色分解）的实际贡献无法从F4/F5中分离。
- **轮数差异**：Agora 100轮 vs B6 5轮（B6+增强20轮部分控制但仅单种子），轮数效应未被充分剥离。

### 方法局限
- **信号集中短端**：空头贡献远大于多头，在多空策略中可行，但长期多头表现平庸，限制了纯多头应用。
- **指标发现仍属常规**：两个涌现指标是标准量化概念（单调性检查、回撤惩罚），未发现真正新颖的指标。
- **提升规则循环**：F5使用训练段夏普作为目标变量与因子优化目标相同，存在循环依赖；其中一个指标在后期相关性下降后仍保持接受状态，规则的鲁棒性有待检验。
- **领域泛化未验证**：SJS框架声称跨领域通用，但论文仅在A股alpha挖掘上验证，缺少其他自主发现任务的实验。

### 统计与可重复性
- **显著性弱**：Agora与最强基线B2的Newey-West p=0.424，无法拒绝无差异的原假设；多数据集的滚动窗口是补充而非替代统计推断。
- **资源消耗高**：100轮主运行需60 GPU小时+4500 LLM API调用，阻碍大规模种子复现。
