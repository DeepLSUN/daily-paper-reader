## 摘要
模型崩溃是从合成数据学习中面临的核心挑战：随着后续代的大型语言模型(LLMs)在越来越多的模型生成数据上进行训练，由于覆盖范围缩小和偏差累积，性能可能会下降。现有研究主要关注如何限制这种退化。然而，在迭代模型演化中，更有意义的目标是确保每个后续模型相比其前身有所改进，这需要在数据策展可操作的粒度上诊断崩溃。我们在指令微调的合成数据自我改进中研究这一问题。我们发现，该场景下的崩溃并非简单的均匀性能退化，而可能表现为能力的极化，即合成训练强化了已有的强技能，同时进一步削弱了薄弱技能。基于这一观察，我们提出了KITE（通过探索进行知识边界指令微调），这是一个两阶段框架，将失败引导的数据生成与边界感知的不确定性策展相结合。在多个数据集和多个开源LLMs上的实验表明，KITE相比强大的合成数据基线能产生更稳定的改进。

## Abstract
Model collapse is a central challenge in learning from synthetic data: as later-generation large language models (LLMs) are trained on an increasing proportion of model-generated data, performance can degrade due to narrowed coverage and accumulated bias. Existing work mainly studies how to bound this degradation. In iterative model evolution, however, the more meaningful objective is to ensure that each successive model improves over its predecessor, which requires diagnosing collapse at a granularity that is actionable for data curation. We study this problem in synthetic data self-improving for instruction tuning. We show that collapse in this setting is not simply uniform performance degradation, but can appear as a polarization of competence, where synthetic training reinforces already strong skills while further degrading weak ones. Motivated by this observation, we propose KITE (Knowledge-boundary Instruction Tuning via Exploration), a two-stage framework that combines failure-guided data generation with boundary-aware uncertainty curation. Experiments across several datasets and multiple open-source LLMs show that KITE yields more stable improvement than strong synthetic-data baselines.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：在迭代指令微调（iterative instruction tuning）中，当模型使用自己生成的合成数据（self-generated synthetic data）进行再训练时，会出现 **模型崩塌（model collapse）**。现有研究主要关注如何限制崩塌幅度，但迭代演化中更有意义的目标是确保每个后续模型比前一个更好，这需要在可操作的数据策展粒度上诊断崩塌。
- **新发现**：作者通过认知诊断模型（DINA）对模型进行细粒度技能分析，发现崩塌并非均匀退化，而是 **能力极化（polarization of competence）**：合成训练会强化模型已经较强的技能，同时进一步削弱本已薄弱的技能。
- **整体意义**：需要一种既能诊断极化又能针对性策展合成数据的方法，实现可持续的自我改进。

## 2. 方法论：KITE框架

提出 **KITE（Knowledge-boundary Instruction Tuning via Exploration）**，两阶段框架：

### 阶段一：失败引导的候选库构建（Failure-Guided Candidate Bank Construction）
- **弱点剖析（Weakness Profiling）**：利用 DINA 认知模型分析模型在训练集上的正确/错误响应模式，推断各技能的掌握程度，提取低掌握技能形成弱点描述符集合 \(S_t\)。
- **排序噪声注入（Rank-based Noise Injection）**：针对每个弱点描述符，使用 **排序噪声解码** 生成候选指令。定义采样分布：
  \[ q_t^{(\alpha)}(v|c) \propto q_t(v|c) \cdot r_t(v|c)^\alpha \]
  其中 \(r_t(v|c)\) 是 token v 在该上下文中的对数几率排名。通过增大低排名 token 的概率，鼓励探索未覆盖区域。
- 生成大型候选池 \(B_t\)（大小 N ≫ 预算 B）。

### 阶段二：边界感知不确定性策展（Boundary-aware Uncertainty Curation）
- **语义不确定性评分（KBU）**：对每个候选指令 x，采样 m 个回答，计算嵌入相似性（RBF 核矩阵 K），并对每个回答按对数似然加权（权重 \(\tilde{w}_i\)），构造加权核矩阵 A，再计算 Rényi-2 熵：
  \[ U_{KBU}(x) = -\log \mathrm{tr}(A^2) \]
- **选择策略**：选择 KBU 值位于特定分位数区间（如 (0.2, 0.8)）的样本——这些样本处于模型语义知识边界附近，既有多种合理回答，又非完全不可信，能提供有用学习信号。
- 选定 B 个样本，由强外部模型（gpt-5-mini）标注正确答案后加入训练集（叠加在原始人类数据上），再从头微调相同基座模型。

## 6. 主要结论与发现

- **模型崩塌的本质**：在迭代指令微调中，崩塌表现为能力极化，而非均匀下降。
- **KITE 的有效性**：在所有 5 个模型上平均性能均优于其他合成方法，且能突破人类数据天花板（Human data）。
- **长时稳定性**：扩展到 9 代后性能单调递增并逐渐饱和，无退化反弹。
- **非推理任务**：KITE 生成的合成数据分布更接近人类数据，迭代训练后困惑度保持稳定，而标准高温合成导致困惑度持续上升。
- **OOD 泛化**：KITE 在未见过的高难度竞赛题上也有提升，表明提升了底层推理能力而非过拟合基准分布。

## 7. 优点

- **细粒度诊断**：利用 DINA 模型识别能力极化，为数据生成提供结构化的弱点指导。
- **创新性双阶段设计**：失败引导生成 + 边界不确定性筛选，直接针对极化的两个方向（强化弱项、抑制过度强化）。
- **理论支撑**：提供了线性高斯模型下的理论分析，证明噪声注入可防止协方差崩塌。
- **实验严谨**：跨多个模型族、多种任务类型、长时演化、OOD 测试、统计检验，证据链条完整。
- **实用性强**：KITE 不增加标注预算，仅重新分配预算到更有效的样本，开销低。

## 8. 不足与局限

- **依赖强外部标签器**：所有方法共用 gpt-5-mini 标注，存在蒸馏因素，KITE 的增益不能完全归因于纯闭环自我改进。
- **DINA 诊断为启发式**：Q 矩阵由 LLM 标记而非专家验证，掌握分数是结构化启发而非校准测量，弱技能选择阈值 (τ=0.30) 为经验设定。
- **模型规模有限**：实验限于 1.7B–8B 参数量的开源模型，未验证更大模型（如 70B+）或 RL 后训练场景。
- **任务覆盖偏重推理**：非推理任务只做了困惑度分析，未在更多创意/知识任务上验证。
- **可解释性不足**：KBU 的语义嵌入使用闭源模型（text-embedding-3-small），内部机制不透明。
- **细微增益可能不显著**：在 MMLU-Pro 和 GPQA 等知识密集型任务上改进较小，部分差异在标准差范围内，作者也谨慎承认。
