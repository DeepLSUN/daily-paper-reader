## 摘要
大型语言模型（LLMs）的高效推理需要在保持模型质量的同时决定计算可以缩减的位置。我们通过多层感知器（MLP）激活稀疏化和令牌级条件路由来研究这一问题。我们首先提出SATS（用于稀疏性的敏感感知阈值校准），这是一种阈值校准方法，使用局部MLP输出敏感度代理来选择逐层门控阈值，而不是直接从激活百分位数校准阈值。虽然SATS保留了通过阈值门控激活来稀疏化MLP激活的现有机制，但它用敏感感知的选择规则取代了基于百分位数的校准。然后，我们引入了一个轻量级的令牌路由框架，该框架在逐令牌的基础上动态选择基础路径或修改路径，而不是将修改后的计算统一应用于所有令牌。我们在多个最新的开源权重LLM上评估了这两种方法。我们的结果表明，在匹配实际稀疏度的情况下，SATS优于基于阈值的稀疏化基线，并且令牌路由相比静态激活修改基线产生了更有利的质量-吞吐量权衡。总体而言，我们的结果表明，改进的阈值校准和令牌路由可以改善LLM中的质量-吞吐量权衡。

## Abstract
Efficient inference in Large Language Models (LLMs) requires deciding where computation can be reduced while preserving model quality. We study this problem through multilayer perceptron (MLP) activation sparsification and token-level conditional routing. We first propose Sensitivity-Aware Thresholding for Sparsity (SATS), a threshold calibration method to choose layerwise gate thresholds using a local MLP output sensitivity proxy rather than calibrating thresholds directly from activation percentiles. While SATS retains the existing mechanism of sparsifying MLP activations by thresholding gate activations, it replaces percentile-based calibration with a sensitivity-aware selection rule. We then introduce a lightweight token routing framework that dynamically selects between a base path and a modified path on a per-token basis, rather than applying the modified computation uniformly to all tokens. We evaluate both methods on multiple recent open-weight LLMs. Our results show that SATS improves over the threshold-based sparsification baseline at matched actual sparsity and that token routing yields a more favorable quality-throughput trade-off than static activation modification baselines. Overall, our results suggest that improved threshold calibration and token routing can improve the quality-throughput trade-off in LLMs.

# 论文详细总结

## 1. 核心问题与整体含义（研究动机和背景）

大型语言模型（LLMs）的推理成本是主要瓶颈，其中多层感知器（MLP）块的计算开销占比很大。现有工作（如CATS）利用MLP激活中的稀疏性，通过对门控激活设定阈值来移除部分神经元，从而加速推理而不改变Transformer架构。但CATS采用基于百分位数的阈值校准方法，只关注移除多少激活值（即稀疏度），而不评估移除神经元对MLP输出造成的实际损害。两个相邻阈值可能带来截然不同的信息损失。此外，现有的激活修改对所有令牌一视同仁，然而同一修改对不同令牌的影响并不均匀。因此，论文旨在解决两个层次的问题：1）如何更精细地选择层间阈值，使得在相同实际稀疏度下模型质量更高；2）如何允许每个令牌动态选择是否使用稀疏路径，以获得更好的质量-吞吐量权衡。

整体含义：通过改进阈值校准方法和引入令牌级条件路由，可以在保持或提升模型质量的同时实现更高效的LLM推理，从而改善模型部署的经济性和实用性。

## 2. 方法论

### 2.1 核心思想

- **SATS（敏感性感知稀疏阈值校准）**：替代CATS的百分位数阈值选择，使用局部MLP输出失真（distortion）作为灵敏度代理来校准每层门控阈值。SATS保留CATS运行时的激活掩码机制，仅改变阈值选择规则。
- **Token Routing（令牌路由）**：为每个令牌单独决定是否使用稀疏路径（应用激活阈值）还是基础路径（完整MLP），路由决策仅基于当前令牌身份，通过预计算查找表实现极低开销。

### 2.2 关键技术细节

#### SATS方法
- 定义局部阈值误差：对于候选阈值 \(t\)，在层 \(\ell\) 上计算密集MLP输出与稀疏MLP输出的相对L2范数误差：
  \[
  \text{Err}_\ell(t) = \mathbb{E}_x\left[ \frac{\|y_\ell(x) - \tilde{y}_{\ell,t}(x)\|_2}{\|y_\ell(x)\|_2 + \epsilon} \right]
  \]
- 建立每层候选阈值集（来自校准数据集的后门控激活分布）。
- 引入每层误差预算 \(b\)，对每层 \(\ell\) 选择满足 \(\text{Err}_\ell(t) \leq b\) 的最大阈值 \(t^*_\ell(b)\)。
- 用外循环搜索预算 \(b\)，使得最终实现的平均稀疏度 \(S(b)\) 接近目标稀疏度 \(s_{\text{tar}}\)。实践中采用粗到细的搜索策略。

#### Token Routing方法
- 在校准语料库中，对每个令牌的出现，分别计算基础路径和稀疏路径下的交叉熵损失，定义损耗差 \(d_i = \max(0, \ell_{\text{sparse}} - \ell_{\text{base}})\)。
- 对每个令牌类型 \(x\)，估计其平均损耗差 \(\hat{s}(x)\)，存入查找表。
- 推理时，根据阈值 \(\tau\) 路由：若 \(\hat{s}(x_i) < \tau\)，走稀疏路径；否则走基础路径。
- \(\tau\) 通过外循环校准，使实现的路由比例 \(R(\tau)\) 接近目标比例 \(r_{\text{tar}}\)。

## 6. 主要结论与发现

- **SATS优于CATS**：在相同实际稀疏度下，SATS在平均下游准确率和困惑度上均优于CATS，尤其在高稀疏度（50%）时改善更明显。
- **令牌路由改进质量-吞吐量权衡**：路由SATS在保持大多数吞吐量增益的同时，其准确率和困惑度优于静态SATS，接近或优于密集基线。例如，Llama 3.1 8B路由50%时，平均准确率从静态SATS的76.2%略降至76.45%（仍接近基线77.11%），但吞吐量提升至35.41 token/s（密集32.12，静态37.08）。Qwen 3路由75%时平均准确率甚至超过密集模型。
- **微调进一步缩小差距**：LoRA微调后，SATS与CATS的差距缩小，但SATS仍保持优势；路由模型在Qwen 3上实现了超越密集基线的平均准确率（75.97→76.31）。
- **训练无关方法即可获得显著改进**：无需额外训练即可提升稀疏模型质量。

## 7. 优点

- **方法创新**：首次将局部MLP输出失真作为阈值校准的信号，更直接地度量损伤，而非仅依赖稀疏度统计。
- **简洁高效**：路由仅利用令牌身份，通过查找表实现，推理开销可忽略（<1%）。
- **实用性**：SATS保持CATS的运行时激活掩码，易于集成到现有推理框架。
- **实验设计严谨**：始终匹配实际稀疏度进行比较，避免了因稀疏度不同导致的性能差异；覆盖多种精度格式和微调设置。
- **可调谐性**：提供参数（预算b、路由阈值τ）允许在不同质量-速度权衡点上灵活部署。
