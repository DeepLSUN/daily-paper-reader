## 摘要
大语言模型（LLMs）展现出卓越的推理能力，但其任务特定微调却以过度自信而臭名昭著，严重阻碍了可信部署。我们提出数据自适应低秩自适应（DALorRA），这是一种简单而有效的变分贝叶斯稀疏框架，将不确定性量化的范式从密集参数空间转移到低秩自适应（LoRA）的轻量级秩级别。基于LoRA本质上聚合了多个可能提供多余模型容量的秩一分量的洞察，DALorRA在秩维度上施加随机掩蔽，从而在训练期间实现模型容量的贝叶斯正则化，并在推理期间实现类似集成的校准。大量实验证明DALorRA在保证推理准确性的同时，具有出色的LLMs校准性能。

## Abstract
Large language models (LLMs) exhibit remarkable reasoning capabilities, but their task-specific fine-tuning is notoriously plagued by overconfidence, severely hindering trustworthy deployment. We propose Data-Adaptive Lower-Rank Adaptation (DALorRA), a simple and effective variational Bayesian sparse framework that shifts the paradigm of uncertainty quantification from the dense parameter space to the lightweight rank level of low-rank adaptation (LoRA). With the insight that LoRA essentially aggregates multiple rank-one components that may provide superfluous model capacity, DALorRA imposes stochastic masking on rank dimensions, enabling Bayesian regularization of model capacity during training and ensemble-like calibration during inference. Extensive experiments demonstrate DALorRA's excellent calibration of LLMs without compromising reasoning accuracy.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：大语言模型（LLMs）在任务特定微调时普遍存在**过自信**（overconfidence）现象，即对错误答案也给出高置信度预测，严重阻碍可信部署。现有不确定性量化方法（如全参数贝叶斯神经网络、深度集成）计算成本极高，无法直接应用于大规模 LLM。
- **现有方法的局限**：
    - 基于输出空间的不确定性（如语义熵）忽略参数级不确定性，对低数据场景下的微调校准不足。
    - 基于参数高效微调（PEFT，特别是 LoRA）的贝叶斯方法（如 BLoB、TFB、C-LoRA）虽然降低计算负担，但仍需在 LoRA 适配器（B 和/或 A）上建模高斯不确定性，引入额外参数和训练开销，且将 LoRA 秩 `r` 视为全局固定超参数，导致在低数据任务中出现容量冗余和过拟合。
- **整体含义**：论文提出一种新型变分贝叶斯稀疏框架 **DALorRA**，将不确定性量化从密集参数空间或 LoRA 适配器空间**转移至更轻量级的秩级别**，通过数据自适应的秩掩蔽实现校准，同时保持 LoRA 的高效性。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 2.1 核心思想

- 标准 LoRA 的权重更新 `∆W = BA` 本质上是 `r` 个秩一分量的组合。若 `r` 过大，某些分量提供多余容量，导致过拟合。
- DALorRA 引入**随机对角掩蔽矩阵** `D = diag(z)`，其中 `z ∈ {0,1}^r`，使更新变为 `∆W = BDA`。通过变分推断学习每个秩分量激活/关闭的后验概率，从而**数据自适应地实现更低秩的适应**，同时捕获结构不确定性。

### 2.2 关键技术细节

1. **概率建模**：
   - 先验：每个掩蔽元素 `z_j` 独立同分布服从伯努利分布 `Bernoulli(p0)`。
   - 变分后验：`q_ϕ(z) = ∏ Bernoulli(z_j; σ(ϕ_j))`，其中 `ϕ_j` 是变分参数，`σ(·)` 为 sigmoid 函数。
   - 目标函数：最小化负证据下界（ELBO）：
     ```
     L(A,B,ϕ) = -E_{q_ϕ}[log p(D|A,B,z)] + KL(q_ϕ(z) || p(z))
     ```
   - 期望项通过蒙特卡洛近似；KL 散度有解析形式。
2. **可微采样**：使用 **Gumbel-Sigmoid 重参数化** 实现离散伯努利变量的梯度估计：
   ```
   z = σ((ϕ + ε)/τ)，其中 ε = log u - log(1 - u), u ~ Uniform(0,1)
   ```
3. **推理**：从学习到的变分后验中采样 `M` 个掩蔽矩阵，平均其预测结果（类似深度集成）。论文设 `M = 10`。
4. **初始化**：令 `ϕ = log(p0/(1 - p0))·1_r`，使初始后验等于先验，避免 KL 爆炸。

### 2.3 算法流程（Algorithm 1）

```
输入：数据集 D，预训练权重 W0，先验概率 p0，温度 τ，样本数 S
输出：B, A, ϕ
1. 初始化 A, B（标准 LoRA 初始化），ϕ ← log(p0/(1-p0))·1_r
2. while 未收敛：
3.   采样 mini-batch
4.   通过 Gumbel-Sigmoid 采样 S 个 z
5.   计算 mini-batch 目标（式(4)），更新 B, A, ϕ
```

## 6. 主要结论与发现

1. **DALorRA 在校准性能上显著优于所有基线**：在 ACC 基本持平甚至更优的前提下，ECE 和 NLL 在 9/10 的设置中获得最好或次好成绩。
2. **学习掩蔽后验至关重要**：同等随机掩蔽即使调整丢弃数也始终劣于 DALorRA，且最优丢弃数因任务而异，而 DALorRA 能数据自适应学习。
3. **充分容量与秩稀疏性结合更优**：增大最大允许秩 `r` 可同时提高 ACC 和校准，说明更大的冗余空间有利于 DALorRA 剪枝。
4. **后验掩蔽模式呈非均匀分布**：不同层、投影矩阵、数据集的后验概率不同，证明数据自适应秩分配的必要性。
5. **极高效率**：仅增加 520 个可训练参数（约 LoRA 的 0.01%），训练时间甚至快于标准 LoRA（因稀疏性降低计算量），整体开销可忽略。

## 7. 优点

- **方法创新**：首次将不确定性量化从参数空间/适配器空间提升至**秩级别**，概念简洁、实现轻量。
- **理论严谨**：严格基于变分贝叶斯 ELBO，无需额外超参数平衡 KL 项（如 C-LoRA 需手动调整 KL 权重），保持理论一致性。
- **集成效应**：推理时对不同掩蔽组合平均预测，天然具备深度集成的鲁棒性，但成本极低。
- **训练加速**：由于随机关闭部分秩分量，正向/反向传播计算量减少，训练速度优于 LoRA 本身。这是难得的“既准又便宜”的方法。
- **实验全面**：覆盖多种基础模型、任务类型、分布偏移、消融分析，结果可重复、结论可靠。

## 8. 不足与局限

1. **任务类型覆盖不全**：仅评估了分类式推理任务（固定答案空间），未涉及开放文本生成（如摘要、对话）中更复杂的不确定性（如幻觉、语义不确定性）。
2. **推理时额外开销**：需采样 `M` 个掩蔽（论文用 M=10）并平均预测，相比单次 LoRA 推理增加了 `M` 倍时间，在延迟敏感应用（如实时交互）中可能成为瓶颈。
3. **秩级不确定性的粒度局限**：DALorRA 仅建模秩分量的开/关不确定性，不学习每个权重的连续不确定性（如 BLoB 中对 A 矩阵建模高斯方差），可能丢失细粒度信息。作者也指出未来可结合权重级贝叶斯方法。
4. **独立伯努利假设**：假设各秩分量、各层、各投影模块的掩蔽相互独立，忽略相关性。文中建议未来研究结构化变分后验。
5. **先验敏感性**：虽然实验显示对 p0 稳健，但理论上最优先验仍可能随任务变化，需要一定经验选择。
