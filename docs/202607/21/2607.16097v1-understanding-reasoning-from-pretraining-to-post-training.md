## 摘要
强化学习已成为提升大语言模型复杂推理能力的核心，但后训练阶段的强化学习通常与先前的预训练孤立研究。因此，两个基本问题仍未解决：（1）预训练选择（模型大小、数据）如何塑造强化学习计算的回报，（2）强化学习实际上对模型做了什么？在标准大语言模型设置中研究这些问题存在困难：预训练语料庞大且不受控，难以将行为归因于预训练还是强化学习，且跨两个阶段的系统计算扫描成本过高。为解决这些挑战，我们使用国际象棋作为受控测试平台，研究从预训练到后训练全流程的推理。我们遵循标准大语言模型训练流程：在人类棋谱上预训练从500万到10亿参数的语言模型，对合成推理轨迹进行监督微调，并在带有可验证奖励的国际象棋谜题上运行强化学习。利用这一框架，我们发现：给定强化学习计算水平下的后训练性能可由预训练损失很好地预测，强化学习奖励曲线的斜率随预训练令牌数量近似线性提升。除规模效应外，我们发现强化学习并非简单锐化监督微调策略：在简单谜题上，它放大了监督微调策略已偏好的正确走法；而在困难谜题上，它浮现出监督微调下几乎不存在的正确走法。我们进一步测试这些发现是否超越国际象棋：在数学领域文本上训练一个10亿参数的语言模型，出现了相同的预测模式——更长预训练的检查点在强化学习后达到更高性能，并在强化学习中提升更快。总之，我们提供了预训练到强化学习接口的量化描述，以及研究从预训练到后训练全流程推理科学的受控测试平台。

## Abstract
Reinforcement learning (RL) has become central to improving large language models (LLMs) on complex reasoning tasks, yet RL post-training is largely studied in isolation from the pretraining that precedes it. As a result, two basic questions remain open: (1) how do pretraining choices (model size, data) shape the returns to RL compute, and (2) what does RL actually do to the model? These questions are difficult to study in the standard LLM setting: pretraining corpora are vast and uncontrolled, making it hard to attribute behaviors to pretraining versus RL, and systematic compute sweeps across both stages are prohibitively expensive. To address these challenges, we use chess as a controlled testbed for studying reasoning across the full pretraining-to-post-training pipeline. We follow the standard LLM training pipeline by pretraining language models from 5M to 1B parameters on human chess games, supervised fine-tuning on synthetic reasoning traces, and running RL on chess puzzles with verifiable rewards. Using this framework, we find that the post-RL performance at given RL compute level is well-predicted from the pretraining loss, and slope of the RL reward curves improves approximately linearly with the pretraining tokens. Beyond scaling, we find that RL does not simply sharpen the SFT policy: on easy puzzles it amplifies correct moves the SFT policy already preferred, while on hard puzzles it surfaces correct moves that were nearly absent under SFT. We further test whether our findings transfer beyond chess by training a 1B language model on math-domain text, where the same predictive pattern emerges: longer-pretrained checkpoints reach higher post-RL performance and improve faster under RL. In sum, we provide a quantitative account of the pretraining-to-RL interface and a controlled testbed for studying the science of reasoning across the full pretraining-to-post-training pipeline.

- **问题**：强化学习（RL）后训练已成为提升大语言模型（LLM）复杂推理能力的关键环节，但预训练与后训练 RL 之间的交互关系尚不明确。两个基本问题悬而未决：

- **背景**：标准 LLM 设置下，预训练语料庞大且不可控，难以分离预训练与 RL 的贡献；系统性的计算扫描（compute sweep）在两个阶段都代价高昂。因此需要一个受控环境来隔离研究。
- **整体含义**：为理解预训练到 RL 的全流程推理提供定量分析和实验范式，进而指导计算资源在预训练和 RL 之间的最优分配。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：以国际象棋为受控测试板，按照标准 LLM 训练流水线（预训练→SFT→RL）训练模型，利用象棋的紧凑动作空间、可验证奖励和丰富的人类数据，系统研究预训练规模与 RL 动态的相互作用。
- **关键步骤**：
  1. **预训练**：在人类棋谱（Lichess 数据）上用下一 token 预测目标训练自回归策略，模型大小从 5M 到 1B 参数（Qwen3 架构，81 词表）。
  2. **监督微调（SFT）**：合成推理轨迹（Chain-of-Thought, CoT），将多个 rollout 合并为前缀树并按深度优先序列化，训练模型在生成推理后选择最佳延续。轨迹来自预训练模型本身（自分布）。
  3. **强化学习（RL）**：在可验证奖励的国际象棋谜题环境中运行 GRPO（Group Relative Policy Optimization）。奖励为二元结果（全部走法正确得 1，否则 0）。
  4. **迁移实验**：在 1B OLMo-2 模型上，使用数学领域文本（Nemotron-CC-Math + Dolma3）预训练，在 GSM8K、MATH、DeepScaler 数据集上 SFT 和 RL，验证发现是否跨领域成立。
- **核心公式/算法**（文字说明）：
  - 本文提出局部对数线性 RL 缩放律：`R_{N,T}(C) = R_ref_{N,T} + B_{N,T} * (log10 C - log10 C_ref)`，其中 `R_ref` 由预训练损失预测，`B_{N,T}`（每 decade RL 性能增益）近似线性依赖于 `log10 T`（预训练 token 数）+ `log10 N`（模型参数）。

  - 联合缩放律：`R(CRL, N, T) = f(L_pt(N,T)) + g(N,T)*(log10 CRL - log10 C_ref)`，结合 Chinchilla 损失预测律形成完全可预测的前沿。

## 6. 主要结论与发现

1. **联合缩放律**：后 RL 性能（pass@1）可由预训练损失准确预测，且预测精度随 RL 计算量增加而提高（Spearman ρ 从 -0.93 升至 -0.99）。RL 奖励曲线的局部斜率（每 decade RL 计算的增益）随预训练 token 数近似线性增加，模型大小提供较弱修正。
2. **计算最优分配**：在固定总计算下，最优预训练 RL 分配中，RL 份额随总计算增加而增加（从 20% 到 28% 在 50M–680M 范围），表明当预算充足时应更多投入 RL，但 RL 仍高度依赖初始化质量。
3. **机制分析**：RL 并非均匀锐化 SFT 策略。
   - 在**简单谜题**上，RL 主要放大 SFT 已偏好的正确走法（Ground-truth amplification）。
   - 在**困难谜题**上，RL 能浮现 SFT 下几乎不存在的正确走法（Tail discovery），但同时也会强化错误走法（Wrong-mode amplification）。
   - 这解释了为何 RL 显著提升 pass@1 但对 pass@16（多次采样）增益有限。
4. **迁移至数学领域**：在 1B 语言模型上观察到相同模式：更长预训练的检查点达到更高后 RL 性能，且 RL 斜率更大，且预训练损失与后 RL 性能的指数关系更强。

## 7. 优点：方法或实验设计的亮点

- **受控测试台设计**：国际象棋提供紧凑动作空间（81 token）、可验证每步奖励、大量人类数据，完美模拟 LLM 流水线但可管理，使得系统扫描预训练和 RL 成为可能。
- **全面的缩放律**：首次将预训练损失与 RL 学习曲线的水平和斜率联系起来，实现了从 (N,T,CRL) 预测后训练性能的端到端公式。
- **细粒度机制分析**：通过重构推理轨迹为前缀树，直接观察策略分布变化、搜索深度/广度演化、走法质量等，揭示了 RL 并非简单的概率锐化，而是包含多种异质效应。
- **跨领域验证**：在数学 NLP 任务上复现主要趋势，证明发现超越棋类游戏，具有普遍性。
- **分析方法丰富**：包括 IsoFLOP 曲线、Chinchilla 律拟合、留一法验证、前沿优化、功率变换拟合、政策分类等，结论扎实。

## 8. 不足与局限

- **领域局限性**：棋盘推理与自然语言推理有本质差异（小词表、离散精确验证、无语言歧义），缩放律的指数和比例不能直接迁移到语言模型，仅揭示结构。
- **计算限制**：模型最大 1B 参数，RL 运行未完全饱和大模型（如 680M 上局部斜率估计易受饱和影响）；前置分析限制在非饱和基准（B3–B4）。更大规模下的趋势需进一步验证。
- **RL 环境限制**：使用唯一解谜题和二元奖励，缺少部分奖励或开放式轨迹，与一些语言任务不同。
- **缩放律的局部近似**：对数线性假设仅适用于局部非饱和区间，极端计算范围（太长或太短）可能失效。
- **合成推理轨迹的固定格式**：特定树序列化格式可能影响 CoT 演化动力学，其他格式（如自然语言链）可能有不同效果。
- **迁移实验仅 1B 模型**：数学领域仅使用单个 1B 模型和有限 RL 计算，统计强度和泛化性有待提升。
