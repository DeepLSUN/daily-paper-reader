## 摘要
全参数微调仍然是适应预训练大型语言模型的强大方法，但它会更新所有权重，且成本高昂。LoRA减少了可训练参数的数量，但并未直接回答在适应过程中应该训练哪些预训练组件、哪些可以冻结。我们提出了TopoTuner，一种拓扑引导的微调框架，用于选择性冻结注意力投影矩阵。该方法将每个投影矩阵视为一个行云，并利用持久性图之间的Wasserstein距离来衡量其在微调过程中拓扑结构的变化。TopoTuner从源数据集中学习一个可复用的冻结配置文件，并将其迁移到域外数据集上高效微调模型，评估特定任务的拓扑漂移是否在问答和情感分析任务中具有泛化性。在LLaMA-3.1-8B、Mistral-7B-v0.3和Qwen3-8B-Base上，TopoTuner仅训练模型参数的1-2%即可与全参数微调相媲美，并在9个模型-数据集设置中的7个中优于LoRA，最多可改变39.57%的投影参数。在最小化更新的同时，TopoTuner相对于全参数微调平均减少20.4%的训练时间，相对于LoRA平均减少5.5%。TopoTuner为可复用冻结配置文件开辟了新方向，使得在一个数据集上学到的微调行为可以在多个任务之间共享。

## Abstract
Full fine-tuning remains a strong way to adapt pretrained LLMs, but it updates all weights and can be expensive. LoRA reduces the number of trainable parameters, but it does not directly answer which pretrained components should be trained and which can be frozen during adaptation. We introduce TopoTuner, a topology-guided fine-tuning framework for selective freezing of attention projection matrices. \method treats each projection matrix as a row cloud and uses Wasserstein distances between persistence diagrams to measure how its topology changes during fine-tuning.   TopoTuner learns a reusable freezing profile from a source dataset and transfers it to efficiently fine-tune models on out-of-domain datasets, evaluating whether task-specific topological drift generalizes across question answering and sentiment analysis tasks.   Across LLaMA-3.1-8B, Mistral-7B-v0.3, and Qwen3-8B-Base, TopoTuner is competitive with full fine-tuning while training only 1-2\% of the model parameters, and outperforms LoRA in 7 out of 9 model-dataset settings, which can change up to 39.57\% of the projection parameters. Along with minimized updates, TopoTuner reduces training time by 20.4\% relative to full fine-tuning and 5.5\% relative to LoRA on average. TopoTuner opens a new direction for reusable freezing profiles, where fine-tuning behavior learned on one dataset can be shared across multiple tasks.

### 1. 论文的核心问题与整体含义（研究动机和背景）

全参数微调仍是适应预训练大语言模型（LLM）的有效方式，但它更新所有权重，计算成本高昂。参数高效微调（PEFT）方法如LoRA减少了可训练参数量，但未回答“哪些预训练组件应该更新、哪些可以冻结”。论文引入拓扑数据驱动（TDA）来度量微调过程中注意力投影矩阵结构重组程度，并据此进行选择性冻结，从而在保持性能的同时大幅降低训练开销，并开创了**可复用冻结配置文件**的新方向——在一个任务上学到的冻结策略可直接迁移到其他任务。

### 2. 论文提出的方法论

- **核心思想**：将每个注意力投影矩阵（K/Q/V/O）的行视为高维点云，用**持久同调（Persistent Homology）**提取其拓扑特征（连通分量、环等），并通过**持久性图**的**Wasserstein距离**衡量微调前后矩阵的结构变化（称为拓扑漂移）。
- **关键技术细节**：
  - 对每个矩阵 \(W_i\)，构建行云 \(X_i\)，计算Vietoris–Rips滤过，得到持久性图 \(\text{PD}_k(X_i)\)。
  - 定义拓扑距离 \(D_i^{(t)} = \text{Wass}_p(\text{PD}_k(X_i^{(0)}), \text{PD}_k(X_i^{(t)}))\)，及漂移 \(\Delta_i^{(t)} = (D_i^{(t)}-D_i^{(t-1)})/D_i^{(1)}\)。
  - **冻结策略**：从源任务（如GSM8K）的全微调检查点计算所有投影矩阵的拓扑漂移，选取漂移最高的 \(b\) 个矩阵（High-drift）或最低的 \(b\) 个矩阵（Low-drift）作为冻结集，在目标任务上直接固定这些矩阵，仅训练其余部分。论文理论证明了当梯度场接近时，该冻结集可稳定复用。
  - **动态早停**：计算所有监控矩阵的平均归一化漂移 \(r(t)\)，当 \(r(t)\leq\eta\) 时停止训练。
- **算法流程**（文字说明）：
  1. 在源任务上全微调预训练模型，保存每个epoch检查点。
  2. 对每个投影矩阵，计算其持久性图，并用Wasserstein距离得到漂移排序。
  3. 按预算 \(b\) 选择High-drift或Low-drift冻结集。
  4. 在目标任务上，冻结该集合中的所有投影矩阵，其余矩阵正常微调；或结合拓扑早停规则动态提前终止。

### 6. 论文的主要结论与发现

- TopoTuner（拓扑高漂移冻结3个投影）在9个模型-数据集设置中的7个上优于LoRA，且仅更新2–3%的参数。
- 性能接近全微调，但训练时间减少20.4%。
- 冻结配置可跨任务迁移：基于GSM8K学习到的冻结集直接用于SST-2、IMDB、MMLU仍有效，且有更好的抗遗忘表现（在QA、SA、IR任务上遗忘更少）。
- 拓扑漂移模式是**模型相关的**：LLaMA和Mistral的V/O漂移大，Qwen的K/Q漂移大，因此不能采用固定规则。
- 拓扑信息揭示的结构变化比纯幅度更丰富，能更好地区分不同任务（如数学推理与情感分析）。

### 7. 优点

- **新颖性**：首个利用LLM权重的拓扑摘要来构建可复用冻结配置的工作，将微调从“参数计数”问题转变为“结构控制”问题。
- **计算效率**：拓扑分析属离线后处理，不增加梯度计算成本；冻结后训练更快、参数更少。
- **可迁移性**：配置文件可跨数据集复用，减少重复探索。
- **理论支撑**：提供了稳定性定理和冻结配置可复用的充分条件。
- **减少过拟合**：高漂移冻结迫使剩余矩阵集中更新，正则化效果好于LoRA和全微调。

### 8. 不足与局限

- **模型访问限制**：需要检查点权重和梯度冻结能力，无法应用于封闭API模型或无法提供内部权重的服务。
- **实验规模**：仅测试了7B/8B级模型，更大模型（如70B/405B）和更多任务族（如多模态）尚未验证。
- **拓扑计算开销**：虽然可并行、可摊销，但对于超大规模矩阵（如MLP层）可能扩展性受限，论文仅分析了注意力投影。
- **静态冻结**：当前冻结集在一次任务上确定后即固定，未探索在训练过程中动态调整冻结的潜力（对比DropBP的动态丢弃）。
- **低漂移策略表现不佳**：论文主要依赖高漂移策略，低漂移策略效果差，说明并非所有“稳定”矩阵都可安全冻结，策略选择依赖经验分析。
