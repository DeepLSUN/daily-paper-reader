## 摘要
基于知识的视觉问答（KB-VQA）要求模型从大规模百科知识库中检索与查询图像匹配的视觉实体，并回答相关问题。现有的多模态检索增强生成（mRAG）方法依赖全局视觉特征来匹配候选实体，但当知识库包含大量视觉相似的实体时，检索器难以区分它们，从而将视觉相似但事实不匹配的干扰项填充到候选集中。由于后续的噪声过滤等处理步骤也被限制在这个固定的候选集中，来自失败检索的错误不可避免地会传播到最终答案。为了解决这些挑战，我们提出了MMAgent-R$^2$，一种智能体mRAG框架，该框架将视觉重排序和主动拒绝作为其内部验证机制。视觉重排序直接比较查询图像和候选图像，捕获文本描述之外的区分性细节，从而在相似候选者中精确识别目标实体；主动拒绝会丢弃不可靠的结果，并在未找到置信匹配时检索额外的候选者，从而超越了固定的候选池。我们设计了一个结合步骤级验证奖励的复合奖励函数，并通过GRPO训练实现了外部检索、内部验证和答案生成的联合优化。在InfoSeek、E-VQA和MMhops上的实验表明，我们所提出的方法达到了最先进的性能，在具有挑战性的检索场景和复杂的多图像多跳推理任务中尤为突出。

## Abstract
Knowledge-based Visual Question Answering (KB-VQA) requires models to retrieve visual entities matching the query image from large-scale encyclopedic knowledge bases and answer related questions. Existing multimodal Retrieval Augmented Generation (mRAG) methods rely on global visual features to match candidate entities, yet when the knowledge base contains numerous visually similar entities, the retriever struggles to distinguish them, populating the candidate set with visually similar but factually mismatched distractors. Since subsequent processing steps such as noise filtering are also confined to this fixed candidate set, errors from failed retrieval inevitably propagate to the final answer. To address these challenges, we propose MMAgent-R$^2$, an agentic mRAG framework that integrates visual reranking and active rejection as its internal verification mechanism. Visual reranking directly compares query and candidate images, capturing discriminative details beyond textual descriptions to precisely identify the target entity among similar candidates; active rejection discards unreliable results and retrieves additional candidates when no confident match is found, moving beyond the fixed candidate pool. We design a composite reward function with step-level verification rewards and achieve joint optimization of external retrieval, internal verification, and answer generation via GRPO training. Experiments on InfoSeek, E-VQA, and MMhops demonstrate that \ours{} achieves state-of-the-art performance, with particularly notable advantages in challenging retrieval scenarios and complex multi-image multi-hop reasoning tasks.

## 2. 方法论

### 核心思想
提出 **MMAgent-R²** 框架，将视觉重排序（Visual Reranking）和主动拒绝（Active Rejection）作为模型的**内部验证机制**，与外部检索工具协同工作。模型被定义为智能体（Agent），通过多轮“推理-行动-反馈”循环自主决策，最终生成答案。整个框架通过强化学习（GRPO）进行端到端联合优化。

### 关键技术细节

- **动作空间（Action Space）** 分为三类：
  - **外部检索**：`SearchImage(j)` 对第 j 张查询图像执行图像检索，返回 Top-K 候选批次（支持连续调用获取非重叠的下一个批次）；`SearchText({q_i})` 执行文本检索，支持并行提交多个查询。
  - **内部验证**：`Rerank(e_k)` 通过视觉比较确认候选实体 e_k 匹配查询图像；`Reject()` 当当前批次无匹配时，主动拒绝并触发下一批次检索，每个图像最多拒绝 `Rej_max` 次。
  - **答案生成**：`Answer(y)` 输出最终答案。

- **智能体推理循环**：模型在每一轮生成 `<reason>...</reason>` 推理内容，然后选择执行一个动作。环境反馈（检索结果）被追加到历史中，形成证据链。循环直到模型输出答案或达到最大轮数 `T_max`。

- **训练方法**：使用 **Group Relative Policy Optimization (GRPO)** 训练。对每个训练样本采样 G 个完整轨迹，并用复合奖励函数优化：
  - **结果奖励 `R_outcome`**：最终答案与 ground truth 匹配得 1，否则 0。
  - **格式奖励 `R_format`**：确保动作输出严格遵循预定义语法。
  - **验证奖励 `R_ver`**：为每个重排序或拒绝决策提供密集的步骤级监督。正确拒绝（当 ground truth 实体不在当前批次中时拒绝）得 δ，正确重排序得 1，否则 0。默认 δ=0.2。

- **候选批次大小**：默认 K=5；最大拒绝次数 `Rej_max=2`；最大交互轮数 `T_max=8`。训练时对环境响应 token 进行掩码，只对模型自身生成的推理和动作 token 进行梯度更新。

- **检索系统**：图像检索使用 EVA-CLIP-8B 编码图像和标题，Faiss-GPU 索引；文本检索使用 E5 编码 Wikipedia 段落（分块 800 字符），返回 Top-3 相关段落。

## 6. 主要结论与发现

- MMAgent-R² 在 InfoSeek (All 50.2%)、E-VQA (All 54.2%)、MMhops Bridging (67.2%) 和 Comparison (39.8%) 上均达到 SOTA，显著优于之前的最佳方法（E-VQA 提升 7.2%，Bridging 提升 13.2%，Comparison 提升 10.4%）。

- 多图像多跳推理场景下优势显著，验证了内部验证机制在复杂任务中的泛化能力。
