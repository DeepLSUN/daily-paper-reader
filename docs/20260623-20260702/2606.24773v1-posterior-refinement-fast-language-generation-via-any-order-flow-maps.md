## 摘要
非自回归生成提供了一种强大多样的迭代精炼范式，使模型能够递归地批评、擦除和重新生成任意的token子集。然而，现有的非自回归模型未能完全实现这一潜力。掩码扩散模型（MDM）存在因子化误差，导致同时生成多个token时样本质量下降。流图语言模型（FMLM）通过联合序列传输绕过了这一瓶颈，实现了出色的少步生成，但牺牲了MDM的推理时灵活性。我们引入FMLM+，该框架通过为FMLM配备掩码式噪声调度来弥合这一差距。在单步生成完整序列的同时，FMLM+后验地评分每个token的全局一致性。利用这一点，我们引入了后验精炼，一种新颖的推理时精炼策略，使模型能够自适应地自我修正输出，以32倍更少的NFE达到离散基线的性能。在多个基准测试中，我们证明带有后验精炼的FMLM+在速度-质量权衡上优于MDM和FMLM两类模型，为高保真语言建模提供了可扩展的基础。

## Abstract
Non-autoregressive generation offers a powerful paradigm for iterative refinement, allowing models to recursively critique, erase and regenerate arbitrary subsets of tokens. However, existing non-autoregressive models fail to realize this potential. Masked Diffusion Models (MDMs) suffer from factorization error, causing sample quality to collapse when generating multiple tokens simultaneously. Flow Map Language Models (FMLMs) circumvent this bottleneck via joint sequence transport for excellent few-step generation, but sacrifice the inference-time flexibility of MDMs. We introduce FMLM+, a framework that bridges this gap by equipping FMLM with masking-style noise schedules. While generating the full sequence in a single step, FMLM+ simultaneously scores the global consistency of each token a posteriori. We leverage this to introduce Posterior Refinement, a novel inference-time refinement strategy that enables the model to adaptively self-correct its outputs, matching the performance of discrete baselines with 32x fewer NFEs. Across diverse benchmarks, we demonstrate that FMLM+ with Posterior Refinement improves the speed--quality tradeoff over both MDM and FMLM families, providing a scalable foundation for high-fidelity language modeling.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：非自回归生成（NAR）理论上支持迭代精炼（递归地批评、擦除、重生成任意 token 子集），但现有两大 NAR 范式均存在致命短板：
  - **掩码扩散模型（MDM）**：存在因子化误差，当同时生成多个 token 时样本质量急剧下降，导致少步生成几乎不可行。
  - **流图语言模型（FMLM）**：通过联合序列传输解决了因子化误差，支持高效的单步或少步生成，但**缺乏 MDM 的推理时灵活性**（例如无法对任意子集进行条件生成或保持已生成 token 不变）。
- **整体目标**：弥合两者差距，提出一个统一框架，既能实现高效的少步/单步生成，又能保留灵活的迭代精炼能力，最终在速度–质量权衡上超越两类基线。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 将 FMLM 的连续流框架与 MDM 的掩码式噪声调度相结合，提出 **FMLM+**。训练时允许部分位置保持干净（作为上下文），其余位置加噪，从而原生支持任意条件模式。
- 利用 FMLM 单步生成完整序列后的输出的**后验置信度**（每个 token 在全局上下文下的拟合程度），设计 **后验精炼（Posterior Refinement, PR）** 迭代策略：保留高置信度 token，重新生成低置信度 token，实现自适应自纠正。

### 关键技术细节
- **训练目标（公式 (10)）**：
  - 定义混合 interpolant：对于上下文位置 C 保持干净（`t=1`），其余位置使用线性插值 `I_t = (1-t)x0 + t x1`。
  - 训练两时间去噪器（TTD）`δ_{s,t}`，使用渐进式自蒸馏损失：
    - 对角项 `L_diag`：监督 `s=t` 时对干净 token 的重建；
    - 离对角项 `L_off`：利用两个较小跳步（`s→u` 和 `u→t`）来蒸馏更大跳步（`s→t`）。
  - 训练时额外对上下文掩码（`m`）以忽略干净位置对损失的贡献。
- **利用预训练 MDM 加速训练**：
  - **蒸馏**：让 FMLM+ 学生匹配 MDM 教师在同一上下文下的 token 概率。
  - **初始化**：直接将 FMLM+ 权重从预训练 MDM 复制（利用 `t=0` 时 FMLM+ 与 MDM 目标等价的性质）。
- **推理（算法 4）**：
  - 初始化所有位置为高斯噪声，设置时间 `s=0, t=1`。
  - 每轮生成候选 `ˆx_1 = δ_{s,t}(x)`，基于 `p_max = max_v (ˆx_1)_{l,v}` 作为后验置信度。
  - 保留高置信度 token（阈值 `p`），其余位置重新注入高斯噪声并继续迭代。
  - 为避免停滞，每轮至少提交 `k` 个 token（回退到 Top-k）。
- **后验置信度的解释**：
  - 理想 FMLM 应输出 one-hot 向量，但实际为软分布。四舍五入误差（如 L1 或 KL 散度）与 `p_max` 单调相关，可作为模型对自身预测的置信度代理。
  - 先验置信度（MDM 或 `δ_{0,0}`）仅基于部分上下文，无法判断多模态下的因子化错误；后验置信度基于完整生成序列，能更准确反映本质不确定性。

## 6. 论文的主要结论与发现

1. **FMLM+ 统一了 FMLM 和 MDM 的优势**，能够同时实现单步生成和任意条件推理。
2. **后验精炼（PR）** 利用 FMLM+ 生成的全局后验置信度进行迭代自纠正，**在 8–32× 更少 NFE 下匹配或超越所有离散和连续基线**，显著改善了速度–质量权衡。
3. **预训练 MDM 可有效加速 FMLM+ 训练**：蒸馏和初始化策略均比从头训练更快收敛且达到更好性能，尤其初始化策略在 GSM8K 上提升约 1.7 倍。
4. **先验置信度在多模态任务（如无条件 Sudoku）上失败**，而后验置信度能有效识别错误 token，实现可靠精炼。
5. **技术洞察**：DiT 的 adaLN 层可将高斯噪声输入映射到 MDM 的 `[MASK]` 表示，从而适配预训练的权重。
